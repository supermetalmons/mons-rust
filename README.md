# mons-rust
`cargo add mons-rust`

or

`npm install mons-rust`

## rules-tests runner

Run all fixtures in `rules-tests`:

`./scripts/run-rules-tests.sh`

Useful options:

`./scripts/run-rules-tests.sh --limit 100`

`./scripts/run-rules-tests.sh --log /tmp/rules-tests.log`

## publishing to npm

`./publish.sh`
